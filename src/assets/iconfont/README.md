# 字体图标列表

以下是本项目中使用的字体图标名称及其对应的类名：

| 图标名称 | 类名                |
| -------- | ------------------- |
| 回到顶部 | `icon-huidaodingbu` |
| 菜单     | `icon-caidan`       |
| 左       | `icon-zuo`          |
| 下       | `icon-xia`          |
| 右       | `icon-you-`         |
| 上       | `icon-shang-`       |
| ...      | ...                 |

## 使用方法

在本项目中，您可以通过以下方式使用这些图标：

```vue
<template>
  <!-- 使用图标作为组件 -->
  <svg-icon iconName="icon-XXX" className="XXX"></svg-icon>

  <!-- 或者直接在模板中使用类名 -->
  <i class="icon-XXX"></i>
</template>
```
